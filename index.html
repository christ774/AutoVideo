Esta aplicación no recolecta datos personales. Solo procesa videos 3D para su publicación en TikTok bajo autorización del usuario.

Política de Privacidad

Última actualización: 12 de marzo de 2026

En ServiTIC, respetamos su privacidad. Esta aplicación ("Vídeo Automático") está diseñada para facilitar la publicación de contenidos 3D generados por el usuario desde servidores remotos hacia la plataforma TikTok.

1. Información que recolectamos

Nuestra aplicación no almacena datos personales en servidores propios. Solo procesamos los archivos de video (.mp4) y los metadatos necesarios (títulos y hashtags) que el usuario decide publicar.

2. Uso de la API de TikTok

Utilizamos el Login Kit de TikTok para autenticar al usuario y la Content Posting API para subir los videos. Estas acciones solo se realizan bajo la autorización expresa del usuario mediante tokens de acceso temporales.

3. Compartir con terceros

No compartimos, vendemos ni distribuimos información de los usuarios a terceros. La única transferencia de datos ocurre directamente hacia TikTok durante el proceso de publicación.

4. Contacto

Para dudas sobre esta política, puede contactar al desarrollador a través de este repositorio de GitHub.
